# Dota Game Root

This folder contains the Dota 2 game-side addon files.

- `scripts/vscripts`: server-side Lua.
- `scripts/npc`: Dota key-value files for units, abilities, and items.
- `resource`: localization.

Current runtime files include the addon entrypoint, forced kobold hero, custom units, abilities/items, services, localization, and CustomNetTables declarations.
