# Dota Addon

This folder mirrors the two major Dota 2 addon roots:

- `game`: server scripts, data files, localization, and gameplay configuration.
- `content`: models, particles, materials, Panorama layout sources, and other content-side files.

The addon now contains a first playable scaffold: game mode entrypoint, custom hero/unit KV, server-side Lua services, and a small Panorama HUD. It still needs a real test map and Workshop Tools log cleanup.
